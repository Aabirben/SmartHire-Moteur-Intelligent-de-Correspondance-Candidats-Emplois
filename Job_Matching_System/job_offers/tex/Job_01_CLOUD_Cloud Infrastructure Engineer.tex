\section*{Job Description}
Join us as a Cloud Infrastructure Engineer to architect and implement cloud solutions. You will design scalable, secure, and cost-effective cloud infrastructure using industry best practices.
\vspace{1em}
\section*{Key Responsibilities}
\begin{itemize}
 \item Monitor and troubleshoot cloud services
 \item Migrate on-premise applications to cloud
 \item Ensure security and compliance in cloud environments
 \item Optimize cloud costs and resource utilization
 \item Design cloud-native architectures
 \item Implement high availability and disaster recovery
 \item Automate cloud infrastructure management
 \item Provide technical guidance on cloud best practices
\end{itemize}
