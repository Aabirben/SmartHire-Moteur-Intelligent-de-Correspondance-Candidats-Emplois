\section*{Job Description}

As a Site Reliability Engineer, you will be instrumental in automating our infrastructure and improving our deployment processes. We need someone who can build reliable CI/CD pipelines and ensure our systems are scalable and secure.

\vspace{1em}
\noindent\textcolor{linecolor}{\rule{\textwidth}{1pt}}
\vspace{1em}

\section*{Key Responsibilities}

\begin{itemize}
  \item Monitor system performance and implement alerting
  \item Implement security best practices and compliance
  \item Document infrastructure and processes
  \item Optimize deployment processes and reduce downtime
  \item Automate infrastructure provisioning using IaC tools
  \item Build and maintain CI/CD pipelines
  \item Troubleshoot production issues and incidents
\end{itemize}
