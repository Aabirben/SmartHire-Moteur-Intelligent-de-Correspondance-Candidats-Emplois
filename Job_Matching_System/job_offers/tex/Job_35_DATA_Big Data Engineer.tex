We are hiring a Big Data Engineer to help us unlock insights from our data. You will design and build data pipelines, develop ML models, and work with large-scale datasets to solve complex business problems.

 \item Develop machine learning models and algorithms
 \item Deploy ML models to production
 \item Process and analyze large datasets

 \item Document data workflows and models
 \item Implement data quality checks and monitoring
 \item Optimize data processing and storage
