\section*{Job Description}
We are seeking a talented Backend Developer to join our engineering team. You will be responsible for building and maintaining scalable backend services that power our applications. This role requires strong problem-solving skills and experience with modern backend technologies.
\vspace{1em}
\section*{Key Responsibilities}
\begin{itemize}
 \item Write clean, maintainable, and well-documented code
 \item Optimize database queries and ensure data integrity
 \item Participate in code reviews and technical discussions
 \item Collaborate with frontend developers to integrate APIs
 \item Implement authentication and authorization mechanisms
 \item Monitor application performance and troubleshoot issues
 \item Design and implement scalable backend services and RESTful APIs
 \item Develop microservices architecture using modern frameworks
\end{itemize}
