\section*{Job Description}
We are hiring a Machine Learning Engineer to help us unlock insights from our data. You will design and build data pipelines, develop ML models, and work with large-scale datasets to solve complex business problems.
\vspace{1em}
\section*{Key Responsibilities}
\begin{itemize}
 \item Develop machine learning models and algorithms
 \item Collaborate with data scientists and analysts
 \item Deploy ML models to production
 \item Document data workflows and models
 \item Process and analyze large datasets
 \item Optimize data processing and storage
 \item Design and build scalable data pipelines
 \item Implement data quality checks and monitoring
\end{itemize}
