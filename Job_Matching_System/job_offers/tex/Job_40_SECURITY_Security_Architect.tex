\section*{Job Description}

We need a Security Architect to protect our systems and data. You will be responsible for security assessments, incident response, and implementing security best practices across our infrastructure.

\begin{itemize}
  \item Conduct security awareness training
  \item Stay updated on latest security threats and trends
  \item Conduct security assessments and penetration testing
  \item Implement security controls and monitoring
  \item Perform vulnerability scanning and remediation
  \item Develop security policies and procedures
  \item Respond to security incidents and threats
