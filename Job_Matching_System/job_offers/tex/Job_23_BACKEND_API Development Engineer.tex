We are seeking a talented API Development Engineer to join our engineering team. You will be responsible for building and maintaining scalable backend services that power our applications. This role requires strong problem-solving skills and experience with modern backend technologies.

\begin{itemize}

 \item Monitor application performance and troubleshoot issues
 \item Optimize database queries and ensure data integrity

 \item Develop microservices architecture using modern frameworks
