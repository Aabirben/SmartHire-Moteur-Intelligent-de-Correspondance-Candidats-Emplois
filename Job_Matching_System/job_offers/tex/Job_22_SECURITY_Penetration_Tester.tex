We need a Penetration Tester to protect our systems and data. You will be responsible for security assessments, incident response, and implementing security best practices across our infrastructure.

\begin{itemize}
  \item Stay updated on latest security threats and trends

  \item Conduct security awareness training
  \item Develop security policies and procedures

  \item Perform vulnerability scanning and remediation
